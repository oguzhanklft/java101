# Burç Bulan Program
***
Java dersi kapsamında koşullu ifadeler yardımıyla kullanıcının girdiği gün ve ay ifadesine bağlı olarak burç bulan program.
***
**TR** -> [Kodluyoruz.org](https://www.kodluyoruz.org/) ve [Patika.dev](https://www.patika.dev/tr) Java eğitimleri kapsamında burç bulan program.
***
**EN** -> The program that finds the user's horoscope within the scope of [Kodluyoruz.org](https://www.kodluyoruz.org/) and [Patika.dev](https://www.patika.dev/tr) Java courses.
